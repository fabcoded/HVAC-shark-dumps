# External Captures

This directory contains protocol frames sourced from community members,
forum posts, and public repositories — not from controlled lab captures.

## Data quality warning

The raw frames in these files may contain transcription errors. The decode
annotations reflect each contributor's interpretation, which may differ
from verified protocol documentation.

Findings from external sources can bear significant value but must always
be cross-validated against independent captures before being adopted into
the protocol spec. Never promote external-capture claims to confirmed
status without discussion and independent proof.
