# HVAC-shark-dumps

This repository contains packet capture dumps from HVAC (Heating, Ventilation, and Air Conditioning) systems that can be analyzed using the HVAC-shark Wireshark dissectors.

## Disclaimer

Please note that the data provided in this repository may contain errors, malformed entries, or lack proper annotations and comments. Users should not rely solely on this data for critical applications. Always validate and verify the information before using it in your projects.

## Purpose

These packet dumps serve as sample data for testing and debugging HVAC communication protocols using Wireshark. The dumps can be opened and decoded with the custom dissectors from the [HVAC-shark project](https://github.com/fabianschwamborn/HVAC-shark/tree/master/wireshark_dissectors).

## Usage

1. Install Wireshark
2. Install the HVAC-shark dissectors from the linked repository
3. Open any .pcap or .pcapng file from this repository in Wireshark
4. The packets should be automatically decoded using the appropriate dissector

## Compatibility

These dumps are meant to be used with the latest version of the HVAC-shark Wireshark dissectors. Please ensure you have the latest version installed for proper packet analysis.
